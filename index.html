Hi. I'm plasma4.

Here are some of my projects:
  • Magix: Magix is a gigantic content mod developed by @pelletsstarPL for NeverEnding Legacy, a game made by Orteil. The original mod creator developed most of the mod but unfortunately stopped working on the project a while back due to IRL stuff.
  • RuntimeFS: a localhost in your browser. It allows you to open HTML projects, use secure encryption, and comes with full offline and data export functionality that no other tool has!
  • LittleExport: A tiny, customizable JS tool that scrapes client-side storage types and converts it to a readable .tar.gz or encrypted file (including IndexedDB/OPFS). Powers RuntimeFS export.
  • FractalSky: a high-quality online fractal viewer, including shading and custom coloring options.
  • HyperNumber: a high-precision googology number algorithm for websites (and used in my games) using BigInt. Uses a unique 2-character number labelling system (so Ut means untrigintillion).
  • AVBD-Web: (name not finalized, in development) A highly performant physics and game engine built for web. Uses the new AVBD physics algorithm. Planned to be developed alongside an infinite, fractal-related mining game!
  • MacroJS: (planned) A small JS-only tool that allows for custom mouse and keyboard macros, with action-recording features and natural input automation.
