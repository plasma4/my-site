Modding:
Note that mods may compromise your privacy and are not validated by me. Please be aware of this when trying to add a mod!

Mods allow you to customize the game. Scripts should be in JavaScript.
You can use the comments in the source code to guide you. Mods will automatically save.
To modify add CSS, change the extraCSS variable. (Note that the CSS code added is added AFTER the game's own style sheet.)
To detect whenever the game updates everything, call Game.addEventListener("save", f). Replace the variable f with a function.
Make sure that the "save" event function returns an array with two strings: [string, string]. The first string should be the ID of your save data (make it unique). The second string should be the save to add. This save data is automatically converted to Base64. Although you can have multiple save data identifiers, you should only have one single function for each.

To modify what happens when the save loads, call Game.addEventListener("load", f).
Replace f with the function that you want to use.
This function should accept the input as an object. This object contains all save data. The keys of the object are from the identifiers you added from the "save" event.
The best way to understand this is to play around with it yourself and test it out!

You may also choose to create additional events. You can input any event name into Game.addEventListener("event name"), and trigger them with Game.triggerEventListeners("event name", data). Note that Game.triggerEventListeners returns an array of the values returned by all functions, and will work on default events such as "load". You can set the data variable to an object, which will get passed to all events added through addEventListener().
You can also set window.hideModWarning to true before the game loads through an extension to skip the mod warning whenever the game loads up.
